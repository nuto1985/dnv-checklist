DNV Checklist App (Replit Hosting)
----------------------------------

1. Go to https://replit.com/ and sign in
2. Create a new Repl: Python (Flask)
3. Upload these 3 files:
   - main.py
   - checklist_data.json
   - requirements.txt
4. Click RUN
5. Your app will launch on a public URL

Access:
- Dashboard: / (home)
- Full Report: /report
